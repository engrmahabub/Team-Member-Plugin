== Description ==
SM Team is a wordpress plugin to list and display team members as grid. You can add team members profile, social links, image, location etc.

= Shortcode =
> `[sm-member theme="theme_name"]`

= Features =
* Fully Responsive.
* Clean Design.
* Use via shortcode.
* Bootstrap based design.
* Social profile link integrated.
* Add Team Member From `SM Team tab`.
* All browser compatible.
* Easy to setup.
* Add Unlimited members.

= Fields =
* Member Name
* Member Details
* Designation
* Member Image
* Email
* Personal Site
* Mobile
* Telephone
* Location
* Linkedin
* Twitter
* Facebook
* Google+
* Youtube
* Vimeo

== Installation ==
1. Upload the entire `sm-team` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

After activate plugin you will see "SM Team" menu at side nav menu bar on WordPress dashboard.

First create some team member from "SM Team".

After creating some member you need to use shortcode on the page where you want to display `Member List`.

= Usage =

1. Go to SM Team->Add New.
2. Add some member.
3. Use shortcede [sm-member] anywhere your page to display Team as grid.
